## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Create .env
###### Rename file .env_example to .env and add credentials

## Development Server


```bash
npm run prisma:init
npm run prisma:migrate
npm run prisma:generate
npm run dev
```

## Production

```bash
npm run prisma:init
npm run prisma:migrate
npm run prisma:generate
npm start
```
